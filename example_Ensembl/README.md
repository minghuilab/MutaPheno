`example_ensembl.csv`: The input file contains Ensembl missense mutations.

`example_ensembl_MutaPheno_FeatureAnnotations.csv`: Contains raw feature annotation data for each mutation.

`example_ensembl_MutaPheno_Predictions.csv`: Contains MutaPheno prediction result (MutaPheno_score and MutaPheno_label) for each mutation.

`example_ensembl_MutaPheno_feature_SHAP.csv`: Contains feature SHAP data for each mutation.
