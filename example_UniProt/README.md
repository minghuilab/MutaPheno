`example_uniprot.csv`: The input file contains UniProt missense mutations.

`example_uniprot_MutaPheno_FeatureAnnotations.csv`: Contains raw feature annotation data for each mutation.

`example_uniprot_MutaPheno_Predictions.csv`: Contains MutaPheno prediction result (MutaPheno_score and MutaPheno_label) for each mutation.

`example_uniprot_MutaPheno_feature_SHAP.csv`: Contains feature SHAP data for each mutation.
