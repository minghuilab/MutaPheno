`example_refseq.csv`: The input file contains RefSeq missense mutations.

`example_refseq_MutaPheno_FeatureAnnotations.csv`: Contains raw feature annotation data for each mutation.

`example_refseq_MutaPheno_Predictions.csv`: Contains MutaPheno prediction result (MutaPheno_score and MutaPheno_label) for each mutation.

`example_refseq_MutaPheno_feature_SHAP.csv`: Contains feature SHAP data for each mutation.
